# üî¨ Noctem Research Agent

Automated research system that uses Warp CLI to systematically explore research questions across STEAM disciplines.

## Overview

The Research Agent is designed to:
- **Continuously research** questions relevant to Noctem's development
- **Use Warp CLI** for AI-assisted research with access to latest information
- **Save findings** regularly to markdown files organized by category
- **Track progress** with automatic state management
- **Generate new questions** dynamically when queue runs low
- **Run autonomously** until credits exhausted or 100 questions answered

## Prerequisites

### Required
- **Python 3.8+** (already have this for Noctem)
- **Warp CLI** installed and authenticated
  - Download: https://www.warp.dev/
  - Authenticate: `warp login`

### Verification
```powershell
# Check Python
python --version

# Check Warp CLI
warp --version
warp chat "test message"
```

## Quick Start

### 1. Start Research Session

```powershell
# From the noctem directory
cd research
python research_agent.py
```

The agent will:
1. Check for Warp CLI availability
2. Load previous progress (if any)
3. Begin researching questions from `questions.json`
4. Save findings to `findings/` directory
5. Generate new questions when queue runs low
6. Continue until 100 questions answered or interrupted

### 2. Monitor Progress

The agent provides real-time console output:

### 3. Stop & Resume

**Graceful Stop:**
- Press `Ctrl+C` - Agent saves state before exiting
- State saved every 5 minutes automatically

**Resume:**
- Simply run `python research_agent.py` again
- Agent loads previous progress and continues

## Configuration

### Files

- **`questions.json`** - Research question queue
- **`state.json`** - Current progress tracking
- **`findings/`** - Research results organized by category

### Adjustable Parameters

Edit `research_agent.py`:

```python
MAX_QUESTIONS = 100              # Total questions to answer
SAVE_INTERVAL_MINUTES = 5        # Auto-save frequency
WARP_TIMEOUT_SECONDS = 300       # Max time per question
```

## Troubleshooting

### "Warp CLI not found"
```powershell
# Install Warp
# Visit: https://www.warp.dev/
# Then authenticate
warp login
```

### "Credits exhausted"
- Research will gracefully stop when Warp API limits reached
- State is saved automatically
- Resume when credits refresh

### "Empty response from Warp"
- Check Warp authentication: `warp chat "test"`
- Verify internet connection
- Question may be too complex - simplify or break into parts

### Research Not Progressing
- Check `state.json` for current question
- Review `questions.json` for pending questions
- Check console output for errors
- Delete `state.json` to reset (loses progress)

## Advanced Usage

### Parallel Research (Not Recommended)
Running multiple instances may:
- Exhaust Warp credits faster
- Create duplicate findings
- Cause state file conflicts

### Custom Warp Prompts
Modify `WarpResearcher.research_question()` to adjust:
- Research depth
- Output format
- Specific focus areas
- Citation requirements

```python
questions = json.loads(Path("questions.json").read_text())

print(f"Completion rate: {len(completed)}/{len(questions)}")
```
